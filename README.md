# Console Games Collection

A collection of classic console-based games implemented in C++.

## Games Included

1. **Snake and Ladders** - A two-player board game with ladders and snakes.
2. **Checkers** - The classic checkers game for two players.
3. **Hand Cricket** - A cricket game simulation played with hand gestures.
4. **Sudoku Solver** - A tool to solve Sudoku puzzles.

## Features

- Interactive console interface
- Colorful display using ANSI escape codes
- Input validation and error handling
- Multiple games in one program

## Prerequisites

- C++ compiler (g++ recommended)
- Windows/Linux/macOS

## How to Compile

### On Linux/macOS:
```bash
g++ Games.cpp -o games
```

### On Windows:
```bash
g++ Games.cpp -o games.exe
```

## How to Run

### On Linux/macOS:
```bash
./games
```

### On Windows:
```bash
games.exe
```

## Gameplay

1. Run the program
2. Choose a game from the menu (1-4)
3. Follow the on-screen instructions for each game
4. Enter 'Y' to play again or 'N' to exit after each game

## Controls

- Use keyboard input as prompted by each game
- Follow the specific instructions displayed for each game

## Note

This program uses ANSI escape codes for colored output, which may not display correctly on all terminals. For best experience, use a terminal that supports ANSI colors.
